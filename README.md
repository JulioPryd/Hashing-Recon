# Hashing-Recon
This is open-source tool for hashing favicon.ico images from website usage for Shodan.io and Fofa.info 

# Usage: 
$ git clone https://github.com/JulioPryd/Hashing-Recon.git

$ cd Hashing-Recon

$ pip install -r requirements.txt

$ python3 hash.py
